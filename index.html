Digital Learning Hub, UNSW Art & Design

Welcome to the IML github, a home for code, documentation and other information relating to the Interactive Media Lab at UNSW Art & Design.

We will be using this page as a hub to link to other locations and repositories where students will be able to find specific code examples, walkthroughs and in-depth tutorials.

Arduino

Arduino is a programmable microcontroller which allows artists and makers to incorporate sensors, motors, lights and other electronics into creative projects.

The IML github hosts walkthrough repositories explaining how to get Arduino set up in conjunction with Max or Processing, in addition to running beginners Arduino workshops during term time. There is also an introductory Arduino badge coming to my.artdesign soon.

Processing

Processing is a free, open source programming environment designed to teach the foundations of computation through creativity. It has excellent documentation, a highly engaged online community, and its functionality can be greatly expanded beyond images and animation through use of libraries. Processing is taught or used in various Media Arts and Design stream modules at UNSW Art & Design, and the IML also runs introductory workshops during term time.

Max

Max, sometimes referred to as "Max/MSP", is a visual programming environment, where objects are connected to one another with patch cables. Each object completes simple tasks, such as combining two numbers, sending trigger outputs, or playing and manipulating audio, which is one of Max's major strengths. Max also has excellent documentation, which is accessible by right clicking on any object and opening its help or reference file.

As above, there are repositories on the IML github for using Max with Arduino, in addition to an example which receives location data from a Kinect via a Processing sketch.

Max is not free. While all lab computers on campus have it installed, if students intend to use it on their personal computers, they must purchase a license through the Cycling74 website. There is a free 30 day trial, but once it has expired you will no longer be able to save your work.
